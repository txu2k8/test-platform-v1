# YAPI同步

列表显示为从YAPI web hook接收到的接口变更事件。

# 操作 - 同步：

同步指定事件对应的接口信息。

  1. 获取事件对应YAPI接口信息详情，如果获取不到，忽略。
  2. 检查本地数据库接口表里该接口是否已存在（查询yapi_id或method+path）
    1. 若存在，diff并更新
    2. 若不存在，新增插入

# 操作 - 删除

软删除

# 顶部工具栏 - 数据同步

全量变更事件处理，同步完成后删除对应事件，添加变更历史到接口信息（可在接口详情->变更历史处查看）。
